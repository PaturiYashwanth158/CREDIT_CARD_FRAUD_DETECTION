# CREDIT_CARD_FRAUD_DETECTION
CREDIT_CARD_FRAUD_DETECTION
An enterprise-grade fraud detection system built using XGBoost, SMOTE, and Flask API, designed to detect fraudulent credit card Transaction

Steps:

1. Place creditcard.csv in backend folder
2. pip install -r requirements.txt
3. python train_model.py
4. python app.py
5. Open frontend/index.html

Includes:
- Random Forest
- XGBoost
- Neural Network (MLP)
- SMOTE Imbalance Handling
- Soft Voting Ensemble
- ROC-AUC Evaluation
